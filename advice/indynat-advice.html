Hi, my name is Natalie

and I have some advice for you!

Make things easy.

Use tools, cheatsheets, W3C and Google!

Hardly anyone codes from scratch.
Invest in a good code editor and know how to use it. It will save you time. The less you are bogged down in the mundane, the more you will learn!

But not too easy.

Have you ever thought about one word, but typed something similar? So much of what we do is habit/muscle memory.
Instead of copying and pasting shorter lines of code, go ahead and type it out.It will get you "thinking and typing" code faster!
